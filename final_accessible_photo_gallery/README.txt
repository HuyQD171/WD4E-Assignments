Final Accessible Interactive Photo Gallery

This project includes:
- index.html
- css/style.css
- js/script.js
- 6 images with alt text
- Mouse support
- Keyboard support
- onload event
- focus and blur events
- automatic tabindex attributes

How to test:
1. Open index.html.
2. Move the mouse over an image.
3. Move the mouse away.
4. Press Tab to focus each image.
5. Press Shift + Tab to go backward.
6. Open DevTools Console to see console.log messages.
